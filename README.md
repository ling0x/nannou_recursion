# nannou_recursion

Generative art built with **Rust + [Nannou](https://nannou.cc)** — a creative
coding framework inspired by Processing and openFrameworks.

Recursion is visualised through the **Koch Snowflake** fractal: starting from a
plain equilateral triangle, every edge is recursively subdivided into four
shorter segments that form an outward spike. Watch the fractal emerge live as
the depth cycles from 0 → 6.
